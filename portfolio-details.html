Portfolio Details

La société Rhévolutions, spécialisée dans le secteur des ressources humaines, a besoin d'une application qui lui permet de suivre les principaux indicateurs de son activité. Elle possède les données mais n'a pas les compétences pour développer un tel produit. Elle a le soucis d'avoir une interface graphique claire et facilement évolutive (qui ne requiert pas un ingénieur logiciel).

Contexte du projet

Vous trouverez dans les ressources le fichier utilisé par la société Rhévolutions pour sauvegarder ses données. La société ne peut pas vous garantir que le fichier ne contient pas d'erreurs et qu'il est correctement rempli.

A l'aide du fichier, le client souhaite voir apparaître sur son tableau de bord les éléments suivants :

  • Nombre d'employés
  • Age moyen
  • Nombre de départements
  • Nombre d'employés par département
  • Nombré d'employés par âge
  • Répartition par sexe
  • Diversité (sexe / origines)
  • Nombre d'employés par état
  • Calcul du taux d'attrition
  • Tendances d'embauches par mois

Il tient à ce que le tableau de bord soit le plus graphique possible (une couleur d'accentuation, des icones, titres, labels etc...).

Pour la conception, il est fortement recommandé d'utiliser une application web nommée "Tableau" (lien en ressource). Elle permet de faire le travail sans avoir d'infrastructures internes.

BONUS 1 : Le tableau de bord doit pouvoir être filtré par différents indicateurs.

BONUS 2 : Concevoir un tableau de bord qui se met à jour via un connecteur comme un connecteur Google Drive.
